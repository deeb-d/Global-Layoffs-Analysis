This project explores global tech layoffs between 2020 and 2026. The goal was to understand which companies, industries, and countries were most affected and how layoffs evolved over time.
Using Python for the data analysis and Tableau for visualization, I built an interactive dashboard to make the trends easier to explore.
What I did:
- Cleaned and explored the dataset using Python and pandas
- Identified companies with the largest layoffs
- Analyzed layoffs by country and industry
- Examined how layoffs changed over time
- Built an interactive dashboard in Tableau
  
Tools I used:
- Python
- Pandas
- Tableau Public

Main things I found:
- Amazon had the highest number of layoffs in the dataset
- Layoffs peaked globally in late 2022, which matches the broader tech downturn
- The United States accounts for the majority of layoffs in the dataset
- Layoffs are heavily concentrated among a small number of large tech companies

Project structure:
